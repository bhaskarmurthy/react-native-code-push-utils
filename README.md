# react-native-code-push-utils

Utilities for CodePush

## Installation

```sh
npm install react-native-code-push-utils
```

## Usage

```js
import CodePushUtils from "react-native-code-push-utils";

// ...

const result = await CodePushUtils.multiply(3, 7);
```
